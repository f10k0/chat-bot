# Чат-Бот на React

Простой чат-бот для демонстрации React.

Было разработано простое React-приложение чат-бота, позволяющее пользователю общаться с виртуальным помощником. Приложение демонстрирует базовые принципы работы с React.

Ввод сообщений - текстовое поле для ввода пользовательских сообщений

Отправка сообщений - возможность отправки сообщений по кнопке или клавише Enter

Автоматические ответы - система ответов бота на основе ключевых слов

История сообщений - отображение всей переписки в течение сессии

Визуальное оформление - различные стили для сообщений пользователя и бота
